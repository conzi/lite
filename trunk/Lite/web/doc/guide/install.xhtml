调试环境如何使用

启动测试服务器后，窗口半透明最前端显示在桌面右侧，您可以选择需要测试的网站目录。选中目录后直接拖放至调试服务器上端。测试服务器自动更新网站目录。

点击部署测试模板，系统将自动将本书的测试用例拷贝到您的网站目录下。 浏览修改吧。。。。
