这是一本什么书

这是一本系统介绍模板编程思想的图书

一提到模板语言，可能很多人并不把它当回事，觉得这个问题太简单了，一种模板语言不就是几个小时看完语法说明就能上手的吗？

是的，模板语言确实应该设计的非常简单，甚至要简单到允许一个非专业的编程人员修改。 不过，简单的结果是需要天才的创意的，如果我们把模板编程的方法做一些系统深入的思考，还是有很多话可以说说。。。

也是一本介绍面向数据的前端编程的图书

前端编程的历史还太短，而被重视的历史就更短，但是这块确是编程环节中非常繁琐也是最为薄弱的一面。 我们前端编程的方法、工具都很原始，当后端工程师在给程序设计更合理的数据模型的时候， 前端工程师还在用最原始的DOM　API 操作冗余的html。或者用js糟糕的字符处理方式生成前端乱糟糟的代码。

这里我们会介绍一种用来改变前端编程体验的前端模板技术。通过它，我们可以在前端实现面向数据的编程。

但更是一本 Lite 模板平台的权威指南

模板技术处在前后端开发的狭缝中，少有人重视，它的发展还很原始，功能太弱，还有很大的发展空间。

而Lite作为模板技术的一颗新星，他有太多值得称道的创新思想，也有更加开放的扩展方法。是一个模板开发与模板语言设计的强大平台。

本书将从多个角度全方位深入解读Lite模板的具体技术与运用，包括基础语法的使用，简单插件的扩展，以及如何在Lite的平台上设计自己全新的模板语言。

只要涉及到web（模板不限于web，但是本书主要讲web的运用场景），难免就要生成一大堆让浏览器解析的标记语言的文本。

那么，如何生成一大堆文本数据呢？我们先举个例子吧，然后八仙过海各显神通。
